# Release Post: AI Game LaunchPad v1.0 Workflow Studio Beta

I built AI Game LaunchPad v1.0, a workflow-based AI studio for turning game ideas into playable HTML5/WebGL prototypes.

Instead of only asking AI for code, the platform turns a prompt into a workflow:

Prompt → AI World Builder → Template Selector → Game Config → Playable Renderer → Launch Page → Share / Feedback.

Features:

- Workflow Studio
- Playable Canvas/WebGL-style prototypes
- AI sandbox editor
- Version history
- Game launch pages
- Reviews and scores
- Share and iframe embed support
- Generation usage limits

Looking for feedback from indie game creators, web developers, and AI tool builders.
